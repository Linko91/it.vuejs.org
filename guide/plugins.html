Plugins

Scrivere un Plugin

I Plugins sono funzionalità inserite a livello globale in Vue. Non ci sono ambiti strettamente predefiniti per i plugin - ecco una lista dei tipici plugin che potresti scrivere:

  1. Aggiungere dei metodo o properità globali. Esempio vue-element

  2. Aggiungere assets globali: direttive/filtri/transizioni. Esempio vue-touch

  3. Aggiungere dei metodi all’istanza Vue legandoli tramite Vue.prototype.

  4. Una libreria che fornisce delle API nuove, magari sfruttando alcune sopra definite. Esempio vue-router

Plugin e Strumenti Disponibili

  • vue-router: Il sistema di routing ufficiale di Vue.js, integrato profondamente con Vue.js per creare Applicazioni a Pagina Singola.

  • vue-resource: Un plugin per permettere di effettuare chiamate asincrone tramite XMLHttpRequest o JSONP.

  • vue-async-data: Plugin per caricare i dati in modo asincrono.

  • vue-validator: Un plugin per la validazione dei form.

  • vue-devtools: Un’estensione per Chrome devtools per fare debug delle applicazioni Vue.js.

  • vue-touch: Aggiungi movimenti touch sfruttando Hammer.js

  • vue-element: Registra elementi personalizzati con Vue.js
